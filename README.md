## CARA INSTALL DI TERMUX
```bash
> termux-setup-storage
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install bash
> pkg install ffmpeg
> pkg install libwebp
> git clone https://github.com/Hyzerr/MD-V3
> cd MD-V3
> yarn
> mc
> ganti session
> node .

## Untuk Sdcard (File Sudah Di Download)
```bash
> cd /sdcard
> cd MD-V3
> yarn
> mc
> ganti session
> node . 
```
